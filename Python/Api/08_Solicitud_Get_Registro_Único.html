¿Qué es CRUD?

CRUD (Create, Read, Update, Delete) es un acrónimo para las maneras en las que se puede operar sobre información almacenada. Es un nemónico para las cuatro funciones del almacenamiento persistente.

¿Qué es nemómino o mnemónico

Los mnemónicos (a veces llamados "dispositivos mnemotécnicos") son herramientas o técnicas de aprendizaje que ayudan a una persona a aprender y recordar información.
