Crear un punto final DELETE API en Flask

¿Qué es DELETE?

Este verbo afecta directamente al recurso al que se hace la petición. Tiene la capacidad de eliminar el elemento y dejar al servidor sin ese recurso.

El método delete elimina la entidad especificada en el objeto de datos raíz.

Sólo se puede eliminar uno a la vez
