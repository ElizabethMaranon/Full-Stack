Crear una acción de actualización con una solicitud PUT en Flask

¿Qué es PUT?

Los llamados PUT en una API se utilizan para modificar información o datos de un servidor. Estas solicitudes son seguras y se envían a través de la URL.

El método PUT actualiza las propiedades de literal y las propiedades de recurso local y suprime las propiedades de recursos local que no estén incluidas en la solicitud.

El método PUT es el que debemos utilizar para operaciones de actualización, para "updates", mientras que el método POST lo utilizaremos en operaciones para crear nuevos elementos, para "create". Utilizaremos PUT cuando queramos modificar un recurso en concreto que ya forma parte de la colección de recursos.
