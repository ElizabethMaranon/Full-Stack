Instalar dependencias base de datos de Flask y funciones API

¿Qué es una dependencia?

Una dependencia de Python es cualquier paquete o distribución que no esté incluido en la instalación base de Apache Airflow para su versión de Apache Airflow en su entorno de Amazon Managed Workflows apra Apache Airflow

¿Qué es ORM?

Los ORM se utilizan en casi todos los lenguajes y marcos populares y lo que hace un ORM es brindarle la capacidad de comunicarse con la base de datos y hacerlo sin tener que escribir código SQL puro.

Es una herramienta que permite mapear y convertir datos entre sistemas orientados a objetos y bases de datos relacionales. Actúa como un puente, permitiendo a los desarrolladores trabajar con objetos y clases en lugar de tablas y SQL directo.

¿Qué es Flask SQL Alchemy?

Es un ORM el cual nos permite trabajar con una base de datos sin tener que conocer y/o dominar SQL.

Hay veces que no es necesario escribir código SQL desde cero, solamente hay que aplicar algunos métodos útiles para ejecutar esas consultas, eso es lo que alquimia SQL permite.

Flask SQL alchemy es simplemente un contenedor de SQL alchemy, por lo que le permite integrar el ORM en la aplicación flask.

¿Qué es Flask-Marshmallow?

Renderizar json api y empaquetar el código. Hace que la representación de los datos de la API JSON sean más fácil para ver en la documentación algunas de las herramientas, para construir modelos propios y crear esquemas, que permiten estructurar esos datos y crear un punto final API

Flask-Marshmallow es una capa de integración delgada para Flask (un marco web de Python) y marshmallow (una biblioteca de serialización o deserialización de objetos) que agrega características adicionales a marshmallow, incluidos campos de URL e hipervínculos para API listas para HATEOAS. También (opcionalmente) se integra con Flask-SQLAlchemy.

¿¿Qué es Marshmallow-SQLAlchemy?

Es una biblioteca ORM/ODM/framework-agnostic para convertir tipos de datos complejos, como objetos, hacia y desde tipos de datos nativos de Python. Serializa y des-serializa nuestros modelos SQLAlchemy de forma que podamos trabajar más fácilmente en formato JSON

Conecta SQL Alchemy y Marshmallow, permitiendo conectarse a ambas herramientas para usarlas en conjunto.

Instalar dependencias

Desde el terminal ejecutar cada instalador
pipenv install Flask-SQLAlchemy
pipenv install flask-marshmallow
pipenv install marshmallow-sqlalchemy
