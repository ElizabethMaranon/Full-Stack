Pipv, gestor de paquetes o bibliotecas

¿Para que se usa?

Cuando instalamos una biblioteca, se instala con la última versión disponible en ese momento.

Cuando hacemos otro proyecto, puede que la versión haya cambiado y sea una más nueva, por lo que la versión anterior será reemplazada por la versión nueva, quedando inutilizada y el programa anterior queda roto.

Para evitar eso, Pipenv se encarga de guardar virtualmente las versiones de los paquetes, conservandolas incluso después de una actualización.

¿Cómo instalarlo?

Abrimos en el navegador Pipenv alli encontrarás las diversas maneras de instalar dependiendo del sistema operativo que tengas.

Para instalar en windows, abrimos el terminal y ejecutamos pip install pipenv

Crear carpeta: mkdir nombre_carpeta Crear Entorno virtual: py -m venv .\[nombre de la carpeta]\ Dar permisos varios: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process Activar el entorno virtual: Ir a la carpeta SCRIPT y ejecutar el file .\activate Ejecutar código: py micodigo.py
