Creamos un archivo python

En python

Creamos variable
 PrimerArchivo = open("Logger.txt", "w+")
 w+ nos permite escribir en el archivo
 open es una función integrada en el lenguaje principal de Python y abrirá el archivo, si no existe lo creará

Escribimos en el arichivo, agregar contenido dinámicamente al archivo
 PrimerArchivo.write("Mi primer archivo")
 Función write() inserta la cadena en el archivo de texto en una sola línea.

Cerramos el archivo
 PrimerArchivo.close()

En consola

Ejecutamos
 python primerarchivo.py y si no recibimos ningún error, todo está correcto

Ejegutamos
 python logger.txt
 Nos imprime Mi primer archivo

Construir un sistema de registro real

PrimerArchivo = open("logger.txt", "w+")
for i in range(100):
 PrimerArchivo.write(f"Estoy en línea {i + 1}\n")
PrimerArchivo.close()

Agregar uno para incrementarlo y que no comience en 0. Cuando llamas al rango, comienza en 0.

Agregar un retorno de carro, es después de que pasen las llaves. \n
