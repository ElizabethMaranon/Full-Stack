¿Cómo instalar MongoDB?

Instalar MongoDB Server (mongod):

1. Descargar de MongoDB Community Server Download
 a. Versión: Current
 b. Plataforma: Windows
 c. Package: msi

2. Instalación
 a. Setup tipo: Completo
 b. Instalar como servicio: No
 c. Copiar dirección Instalación:
  C:\Program Files\MongoDB\Server\7.0\data\
  C:\Program Files\MongoDB\Server\7.0\bin\
 d. Instalar MongoDB Compass: Si

Instalar MongoDB Shell (mongosh)

1. Descargar de MongoDB Shell Download
 a. Versión: Default
 b. Plataforma: Windows
 c. Package: msi

2. Instalación
 a. Copiar dirección Instalación:
  C:\Users\eliza\AppData\Local\Programs\mongosh\

Crear directorio DB, si no ha sido creado

 cd c:\
 md "\data\db"

Ejecutar

1. Abrir Variable de entorno search
2. Click Variable de entorno
3. Seleccionar Path
4. Click Edit
5. Click New
6. Pegar dirección instalación mongod
 C:\Program Files\MongoDB\Server\7.0\data\
 C:\Program Files\MongoDB\Server\7.0\bin\
7. Pegar dirección instalación mongosh
8. Click OK, OK, OK
9. Cerrar el terminal y reabrir con el nuevo variable de entorno

En el terminal ejecutar
mongod

En nueva ventana separada ejecutar
mongosh
