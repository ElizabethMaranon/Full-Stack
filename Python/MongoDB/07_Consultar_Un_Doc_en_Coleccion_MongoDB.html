Cómo consultar documentos específicos en una colección de MongoDB

Consulta un documento en específico

Para buscar uno en específico usar la clave-valor, en este caso {nombre: "Sylvania"} que al ser un objeto tiene que ir entre llaves.
db.música.find({nombre: "Sylvania"})

Para que la búsqueda ofrezca visualización bonita usar pretty()
db.música.find({nombre: "Sylvania"}).pretty()

Si la colección tiene varios documentos con el mismo nombre, te devolverá todos los que coincidan con la búsqueda>

Equivalente en SQL

SELECT * from colección WHERE name = "documento"
