Diccionario

db.collection.insertMany() Insertar muchos documentos a la vez, usaremos los corchetes para definir cuando acaba y las llaves para delimitar cada objeto, importante usar coma cada vez que se acaba un objeto o clave-valor si se va a añadir otro dato más

Un valor hash es un valor numérico de longitud fija que solo identifica datos. Los valores hash representan grandes cantidades de datos como valores numéricos mucho menores, por lo que se usan con firmas digitales. Puede firmar un valor hash de forma más eficaz que un valor mayor.
