Cómo consultar todos los documentos de una colección MongoDB con el método find()

Función find

db.música.find()

formato más agradable
db.música.find().pretty()

Cada documento creado, obtiene una id automática por parte de Mongo

Equivalente a SQL

SELECT * from música
