Cómo crear una colección MongoDB

Diccionario

Una Base de datos es una recopilación organizada de información o datos estructurados, que normalmente se almacena de forma electrónica en un sistema informático
En MongoDB, las bases de datos contienen una o más colecciones de documentos

Las Colecciones son agrupaciones de documentos. Son equivalentes a las tablas en bases de datos relacionales pero NO nos imponen un esquema o estructura rígida para guardar información. Los Documentos son registros dentro de las colecciones.
MongoDB almacena documentos en colecciones. Las colecciones son análogas a tablas en bases de datos relacionales.
