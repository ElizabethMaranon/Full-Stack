¿Qué es una Api?

API significa interfaz de programación de aplicaciones y es un gran acrónimo que en realidad simplemente significa que es una forma en la que podemos comunicarnos con una aplicación y podemos hacerlo sin implementar algo como un raspador o algo como esto.

Nos brinda un conjunto de comandos que podemos usar y podemos hacer que nuestra aplicación se comunique con otro servidor en otra aplicación

Lo que una API le brinda es un conjunto de puntos finales.

¿Qué son puntos finales?

Un punto final es solo una URL que nos devolverá datos JSON
