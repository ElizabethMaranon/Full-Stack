Guía de instalación de Postman

¿Qué es Postman?

Postman, es una aplicación muy útil que te permite comunicarte con API externas. Una API es un servicio parecido a un sitio web o servidor con el que puedes comunicarte, pero en lugar de recuperar una página web, obtienes datos.

Esta es una forma muy útil de crear aplicaciones y pasar datos de un lado a otro y no es específico de un lenguaje de programación ni de un marco ni nada por el estilo

Postman es una popular herramienta utilizada para probar APIs, permitiendo a los desarrolladores enviar peticiones a servicios web y ver respuestas.
