¿Qué es isinstance

isinstance() nos devuelve si una variable es una instancia, es decir, si hereda de un determinado tipo. En ciertas ocasiones el comportamiento puede ser idéntico a type() , pero cuando trabajamos con herencia, esto deja de cumplirse.

Isinstance se utiliza para verificar si un objeto es una instancia de una clase o una tupla de clases. Devuelve True si el objeto es una instancia de la clase indicada y False en caso contrario.
