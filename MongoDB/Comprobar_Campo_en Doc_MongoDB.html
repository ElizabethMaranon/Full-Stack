Cómo comprobar si existe un campo en un documento MongoDB

Esto es algo necesario debido a lo flexible que es la base de datos Mongo, se pueden añadir documentos con diferentes atributos o campos.

Verificar campo en documento

Añadir documento nuevo con atributo nuevo
db.música.insertOne(...) Reseñas

Para verificar las que SI tengan el atributo
db.música.find({ reseñas: { $exists: true } })

Para verificar las que NO tengan el atributo
db.música.find({ reseñas: { $exists: false } })
