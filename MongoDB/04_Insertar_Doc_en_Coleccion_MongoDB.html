Insertar documentos en una colección MongoDB

Siempre que se trabaje con Mongo se usara db.objeto, que hace referencia a la base de datos

Mongo no usa esquema, no importan los nombres de las columnas o los nombres de las claves, se puede escribir y hacer que cada documento sea tan único como sea necesario.

Nivel de responsabilidad: Mongo no se preocupa por los datos, pero cuando quieres hacer una consulta y se han cambiado los nombres o claves, los datos pueden ser incorrectos.

Al ser un objeto se pasará entre llaves y será un conjunto de pares clave-valor

Añadiremos el nombre del grupo y la fecha publicación

Se puede pasar colección dentro del documento, hay que pasarla como matriz, pasamos canciones

Se puede eliminar una de las canciones y sigue sin haber problema, agrega documento
